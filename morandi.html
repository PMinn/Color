Morandi Colors

  1. Include Morandi Colors cdn in your website like this:
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/PMinn/Color.css/src/Morandi.css">
  2. find the color you need

    		#A1A799 SAGE
  3. Use it:
    like:
    <div style="width:50px;height:50px;background-color:var(--IVORY);">
    or
    .dom {
    background-color: var(--IVORY);
    }
